Clase 5 cómo subir un proyecto en Github

Pasosen local, cómo crear un proyecto desde la terminal

  1. Abrir la termina
  2. Crear una carpeta del proyecto (mkdir "nombre_de_la_carptea")
  3. Entrar a la carpeta del proyecto (cd "nombre_de_la_carptea")
  4. Creamos un archivo index.html (touch index.html)
  5. Iniciar un repositorio dit local (git init)

Pasos en Github cómo crear un nuevo repositorio

  1. Entrar a github
  2. Crear un uevo repositorio
  3. Copiar la url del repositorio
  4. Volver a la terminal
